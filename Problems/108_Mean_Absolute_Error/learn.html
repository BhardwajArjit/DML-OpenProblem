Mean Absolute Error (MAE) Calculation

The Mean Absolute Error (MAE) is a metric used to measure the average absolute difference between predicted and actual values in a dataset.

Mathematical Formulation

MAE is defined as:

\[ MAE = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i| \]

  • \( n \): Total number of data points
  • \( y_i \): Actual value at index \( i \)
  • \( \hat{y}_i \): Predicted value at index \( i \)
  • \( |y_i - \hat{y}_i| \): Absolute difference between actual and predicted values

Implementation Steps

  1. Compute the absolute difference \( |y_i - \hat{y}_i| \) for each data point.
  2. Sum all absolute differences.
  3. Divide the sum by the total number of data points \( n \).

Example Calculation

Given the actual values and predicted values:

  • Actual values: \( y = [3, -0.5, 2, 7] \)
  • Predicted values: \( \hat{y} = [2.5, 0.0, 2, 8] \)

Step-by-step calculation:

  • Compute absolute differences: \[ |3 - 2.5| = 0.5, \quad |-0.5 - 0.0| = 0.5, \quad |2 - 2| = 0, \quad |7 - 8| = 1 \]
  • Sum of absolute differences: \[ 0.5 + 0.5 + 0 + 1 = 2 \]
  • Compute MAE: \[ MAE = \frac{2}{4} = 0.5 \]

The Mean Absolute Error (MAE) for this example is 0.5.
